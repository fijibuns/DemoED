DemoED
==================

Heya :) Fijibuns here! 

I named it DemoEd because it's like a shorter term for Demographic Education....pretty witty..right? lol Anyway

This repository contains a collection of Philippine-focused demographic and macroeconomic data analysis projects built using Python and statistical research methods. The projects explore how economic indicators, labor markets, demographics, and social trends interact to shape economic conditions and investment outlooks in the Philippines.

Each project is designed to transform raw economic and demographic data into meaningful insights through data analytics, visualization, forecasting, and statistical modeling.

## What youll find here 
- Philippine demographic and macroeconomic research projects
- Youth unemployment, inflation, and GDP trend analysis
- Exploratory data analysis (EDA) and statistical interpretation
- Time-series analysis and forecasting models
- Correlation and regression-based economic studies
- Data visualizations and economic dashboards
- End-to-end workflows from data collection to research reporting
